📈 Broker API — R StocksTrader (RoboForex)
Paste your access token from stockstrader.com → Settings → API. Token is stored only in your browser's localStorage and sent straight to api.stockstrader.com with no intermediate proxy.
⚠ Browser CORS note: StocksTrader's API was built for server-side bots and may reject direct browser calls with a CORS error. If Test fails with a CORS / Failed-to-fetch message, install a one-origin CORS extension (Chrome / Firefox) and enable it for api.stockstrader.com only, or proxy through a tiny local server. Never send this token through a public CORS proxy — it gives anyone full trading access to your account.
